# QGnomePlatform-qt6

# About this build
* This repository (the tar.zst archive) includes all three (QGnomePlatform, QAdwaitaDecorations, Adwaita-Qt) parts of the QGnomePlatform / FedoraQt suite that stopped being available / developed after F41.
* This is built on F43, on Qt6, and the code updated (where hneeded) to work with Qt6.
* The release has prebuilt binaries that can be installed using `sudo make install` while inside the `build` directory.
